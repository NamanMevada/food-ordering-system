🛠️ Tech Stack
PHP · MySQL · Bootstrap 5 · JavaScript · XAMPP

👥 3 Role-Based System
🔹 Admin — Approves restaurants, manages users and categories
🔹 Restaurant Owner — Adds menu items with images, updates order status
🔹 Customers — Browses restaurants, adds to cart, places orders & tracks status

⚙️ Key Features
✔️ Secure authentication using PHP Sessions
✔️ Restaurant approval workflow
✔️ Menu management with image upload
✔️ Dynamic cart with quantity control
✔️ Real-time order status tracking
✔️ SQL Injection protection

📚 What I Learned
Database design using foreign keys, complex SQL JOIN queries, role-based access control, and real-world business logic implementation.
